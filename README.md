# Common Ground : Le Github des Passions

**Common Ground** est une plateforme de co-construction massive pour projets locaux, sportifs et créatifs. Elle transforme le réseautage passif en collaboration active grâce à une infrastructure héritée du développement logiciel (Blueprints, Kanban, Matching).

## Vision
Le projet s'attaque à la "Nécessité Collaborative". Dans un monde sur-connecté mais sous-engagé, Common Ground fournit les outils pour assembler des équipes basées sur le temps réel et les compétences critiques.

## Fonctionnalités Clés (Prototype V1)
- **Project Blueprint** : L'ADN du projet en format JSON, définissant les besoins en "Stack" humain.
- **Micro Kanban** : Gestion agile simplifiée pour les non-professionnels.
- **Availability Matcher** : Algorithme de score pondéré (Alpha/Beta) pour le matching profil/projet.

## Tech Stack
- **Core** : TypeScript / Vite
- **UI** : Vanilla CSS Premium Design System
- **Deployment** : CI/CD GitHub Actions

## Installation
```bash
npm install
npm run dev
```
